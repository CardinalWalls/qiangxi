抢戏(qiangxi)是一类语音驱动的 AI agent 体验:AI 在台前自然地演一个角色,你用语音跟它互动——而平时藏起来的那个掌控全局的"导演脑",被实时摊在屏幕上。

范式

脑 / 嘴 分离

一个"导演脑"掌握状态与策略、做所有决定;前台的"嘴"只管自然地把这一步演出来。行为逻辑不塞进表演模型。

交互

语音优先

你开口说,它实时回应。压迫感、节奏、即兴,都在语音里发生,而不是打字框。

透明

编排被看见

它此刻在用哪招、走到哪个阶段、危险多高、你守得怎样——全部实时投影到屏幕,把隐藏的"编排脑子"摊开。

用这套范式做的东西

同一个"脑/嘴分离 + 编排可视化",落到不同场景

主打 · 训练工具

裁了吗

实时语音裁员谈话陪练。你跟一个冷面 HR 真人语音对话,它按套路一步步逼你当场让步;屏幕实时显示它在用哪一招、走到哪个阶段、你守权守得怎么样。一个防御性的反 PUA 训练工具。

LiveKit + Gemini 原生音频导演脑 = 状态机 + 战术库R1–R5 实时打分
